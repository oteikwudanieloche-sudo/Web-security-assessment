. Phase 1: Lab Setup & Network Validation

. Objective
The objective of this phase was to build and validate an isolated cybersecurity testing environment using Kali Linux and Metasploitable2.

. Activities Performed
- Configured virtual machines inside VirtualBox
- Established isolated network communication
- Verified IP addressing
- Validated attacker-to-target reachability
- Confirmed successful ICMP communication

. Outcome
A functional penetration testing lab environment was successfully deployed. Both systems communicated properly over an isolated network, preparing the environment for reconnaissance and vulnerability assessment activities.

. Skills Demonstrated
- Virtualization
- Linux networking
- Lab architecture
- Network validation
- Environment preparation

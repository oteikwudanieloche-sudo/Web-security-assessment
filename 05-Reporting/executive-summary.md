Executive Summary

A security assessment was conducted against a controlled laboratory environment using Kali Linux and Metasploitable2.

The assessment included reconnaissance, service enumeration, vulnerability scanning, and vulnerability validation activities.

Several security weaknesses were identified, including exposed network services, accessible SMB resources, anonymous FTP access, and outdated software components.

These weaknesses increase the attack surface of the target system and may allow unauthorized users to gain access to sensitive resources or leverage known vulnerabilities.

Recommendations include disabling unnecessary services, restricting anonymous access, applying security updates, enforcing authentication controls, and implementing continuous monitoring.

Overall Risk Rating: Medium

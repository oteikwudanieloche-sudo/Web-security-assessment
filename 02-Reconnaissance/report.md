. Reconnaissance & Enumeration

. Objective
The objective of this phase was to identify exposed services, open ports, and operating system information on the target machine using reconnaissance and enumeration techniques.

. Activities Performed
- Verified host reachability using ICMP
- Conducted TCP port scanning using Nmap
- Identified running services and versions
- Performed operating system fingerprinting
- Executed aggressive enumeration scans

. Findings
Multiple network services were identified including FTP, SSH, HTTP, SMB, and MySQL. Several services appeared outdated and potentially vulnerable.

. Security Impact
Exposed services increase the target attack surface and may provide attackers with multiple entry points for exploitation.

. Skills Demonstrated
- Network reconnaissance
- Service enumeration
- OS fingerprinting
- Attack surface identification
- Vulnerability discovery preparation
